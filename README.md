# Password Strength Analyzer

A Python-based tool that evaluates password security using common cybersecurity best practices.

---

## Features

- Checks password length  
- Evaluates character diversity:
  - Uppercase letters  
  - Lowercase letters  
  - Digits  
  - Symbols  
- Categorizes passwords as **Weak**, **Moderate**, or **Strong**  
- Designed as a foundational component for authentication systems  

---

## Why This Matters

Weak passwords remain one of the most common security vulnerabilities.  
This tool simulates how real-world systems enforce stronger credential policies and helps users understand what makes a password secure.
